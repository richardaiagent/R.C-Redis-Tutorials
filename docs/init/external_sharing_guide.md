## 🚀 5분 만에 시작하기

### 1단계: 저장소 확인
```
https://github.com/richardaiagent/R.C-Redis-Tutorials
```

### 2단계: 로컬에 복제
```bash
git clone https://github.com/richardaiagent/R.C-Redis-Tutorials.git
cd R.C-Redis-Tutorials
```

### 3단계: Docker로 실행
```bash
docker-compose up -d
```

### 4단계: Redis 연결 확인
```bash
docker exec -it redis redis-cli ping
```
